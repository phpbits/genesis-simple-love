== Description ==

<strong>This plugin requires Genesis Simple Share</Strong>

Are you using Genesis Simple Share on your <a href="http://goo.gl/rL2pPZ">Genesis Framework</a> Powered Website? Add this feature upgrade to get more engagement with your visitors. Give them an option to 'love' the posts or pages they like. This will give you insights on how many of your visitors actually like a post on your blog.

<strong>Available Filters:</strong><br />
genesis_simple_love_text - Use this filter if you want to change the "Love" text.<br />
genesis_simple_love_message - Use this filter if you want to change the "Thank You for loving this!" alert message.<br />
genesis_simple_loved - Use this filter if you want to change the "You already loved this. Thanks!" alert message.<br />

This plugin uses cookie to check if the user already loved the post. Thanks!
